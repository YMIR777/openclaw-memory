# 个人财务记账应用 - 开发文档

> 记录日期：2026-04-06

## 项目概述
个人记账网页/桌面应用，服务于陪玩收入的追踪和日常开销管理。

---

## 核心需求

### 收入端
- **多平台收入**：支持比心等多个平台，每个平台独立记录
- **双渠道收入**：平台到账（已抽7成）+ 微信转账（工作小号）
- **记录维度**：日期、平台名称、到账金额、老板名称（方便联系）
- **微信小号方案**：用微信小号作为工作号，收入几乎100%纯净

### 支出端
- **快速记账**：支持手动输入，自动识别分类
- **分类体系**：餐饮/交通/购物/娱乐/日用/居住/通讯/医疗/学习/人情/其他
- **支持手动调整分类**

### 资产追踪
- **多账户余额**：银行卡/支付宝/微信/现金等分别记录
- **自动汇总总资产**
- **初始余额**：从某个时间点开始记录（具体待定）

### 数据展示
- **陪玩工作专属界面**：独立 dashboard 展示各种收入数据图表
- **图表类型**：待定（需要参考网站确认风格）

## 技术栈决策

| 类别 | 选择 | 备注 |
|------|------|------|
| 前端框架 | React + TypeScript | pretext 官方支持 React |
| UI 组件库 | shadcn/ui | 设计优雅，完全可定制 |
| 样式 | Tailwind CSS | 快速开发 |
| 图表 | Recharts | React 集成好 |
| 状态管理 | Zustand | 轻量，适合个人项目 |
| 打包（未来） | Tauri | 轻量，比 Electron 小10x |
| 文字动效 | pretext | WebGL/GLSL 驱动的炫酷效果 |
| 字体 | 汉仪书宋二S | 用户指定，已安装 |

### 技术栈局限性说明
- React 生态成熟，shadcn/ui 效果不输原生 app
- Tailwind "轻量"不等于"功能受限"，而是开发效率高
- Recharts 对财务常用图表（饼图/折线图/柱状图）完全够用
- Zustand 对个人项目绑绑有余

## 平台限制

### 比心平台
- **API 导出**：无公开 API，无法自动拉取数据
- **解决方案**：OCR 截图识别 或 手动录入
- **抽成**：平台抽3成，到手7成，直接记录到账金额
