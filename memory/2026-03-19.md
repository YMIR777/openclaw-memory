## StarUML 自动化方案

Claude Code 研究了 StarUML 自动化，发现两种方案：

### 1. StarUML CLI（仅 v5.x+）
```bash
$ alias staruml='/Applications/StarUML.app/Contents/MacOS/StarUML'
$ staruml --help
```

### 2. staruml-controller 扩展（推荐）
- 仓库：https://github.com/pontasan/staruml-controller
- 安装：克隆到 `~/Library/Application Support/StarUML/extensions/user/staruml-controller`
- 启动：Tools > StarUML Controller > Start Server...（默认端口 12345）

**API 示例（用例图）：**
- `POST /api/usecase/diagrams` - 创建用例图
- `POST /api/usecase/actors` - 创建 Actor
- `POST /api/usecase/use-cases` - 创建 Use Case
- `POST /api/usecase/associations` - 创建关联线
- `PUT /api/views/:id` - 调整位置（left/top/width/height）

**用途**：程序化生成 UML 图、自动化用例图设计等
