### 经验沉淀

**PlantUML 类图语法（简洁版，无报错的写法）：**
```plantuml
@startuml
skinparam classAttributeIconSize 0
hide empty members

class Shape {
    - x: number
    - y: number
    - color: string
    + draw(ctx): void
}
class Circle {
    - radius: number
    + draw(ctx): void
}
class Rectangle {
    - width: number
    - height: number
    + draw(ctx): void
}
class Point {
    + x: number
    + y: number
}

Shape <|-- Circle : 继承
Shape <|-- Rectangle : 继承
Shape ..> Point : 依赖
Circle ..> Point : 依赖
Rectangle ..> Point : 依赖
@enduml
```
- 不要在类定义后面直接加 `note right of Shape::method`（1.2026.2 有时会报错）
- 关系标签用 `: 标签 (英文)` 而非纯中文，避免编码问题

**Canvas 拖拽绘图定位算法（已验证正确）：**
```
起点 (x1, y1)，终点 (x2, y2)
- 图形中心 = ((x1+x2)/2, (y1+y2)/2)
- 圆形半径 = distance(起点, 终点) / 2
- 矩形宽高 = |x2-x1|, |y2-y1|
```
