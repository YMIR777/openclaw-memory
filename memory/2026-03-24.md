## 重大事件：Gateway SIGTERM 循环问题

Claude Code 深度排查并解决了两个关键问题：

### 1. Watchdog 导致 Gateway 无限重启
- 自定义 watchdog（`com.ymir.gateway-watchdog.plist`）设计缺陷
- 每 30 秒检查 health，失败就重启，没有考虑 startup-grace
- **解决**：禁用 watchdog，使用原生 launchd 服务

### 2. openclaw vs openclaw-cn 混用问题
- 之前错误地将 openclaw 升级到 2026.3.22
- openclaw npm 包不包含 control-ui
- 协议不匹配导致 WebSocket 1008 错误
- **解决**：回退到 openclaw-cn v0.1.8-fix.3

### 经验总结（已写入 MEMORY.md）
- openclaw-cn 是独立分支，不能与 openclaw 混用
- 所有路径、配置都要明确是哪个版本
- OpenClaw 有内置 health-monitor，不需要外部 watchdog

### QMD 配置详细经验（已沉淀 Obsidian）
- 新建笔记：VibeCoding/05-QMD配置笔记.md
- 核心问题：Node.js fetch 不走代理，qmd hf_transfer 也不走代理
- 解决：patch getRemoteEtag 函数用 curl 替代 Node.js fetch，走代理下载模型
- 关键：HuggingFace HEAD 请求返回 302 重定向到 CDN，最终响应无 ETag，应用 x-linked-etag
- 模型：embeddinggemma（313MB）+ Qwen3-Reranker（610MB）+ qmd-query-expansion（1.2GB）= 2.1GB
- 首次 Metal 加载需要 3-5 分钟 GPU shader 编译，这是正常的，不是卡住
- searchMode "basic" 是错的，可选值只有 query/search/vsearch
- 子会话可绕过主会话 exec 限制（独立 session 运行）
