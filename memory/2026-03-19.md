## 晚上补充3（21:49）

### OpenClaw ACP + Claude Code 集成关键配置（已修复）
需要在 `~/.openclaw/openclaw.json` 添加：

```json
"plugins": {
  "entries": {
    "acpx": {
      "enabled": true,
      "config": {
        "permissionMode": "approve-all",
        "nonInteractivePermissions": "fail"
      }
    }
  }
},
"acp": {
  "enabled": true,
  "backend": "acpx",
  "defaultAgent": "claude",
  "allowedAgents": ["claude", "codex", "pi", "opencode", "gemini", "kimi"],
  "maxConcurrentSessions": 8
}
```
