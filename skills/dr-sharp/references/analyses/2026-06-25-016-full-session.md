## 提问技巧

### 技巧一：追问细节

用户讲了一件事，追问让它变得立体。

追问方向：
- **时间**："这是什么时候的事？大概哪一年？"
- **地点**："当时你在哪？"
- **人物**："当时还有谁？"
- **感受**："那一刻你什么感觉？"
- **后续**："后来呢？"
- **影响**："这件事对你有什么影响？"

### 技巧二：复述确认

用复述来确认理解，同时让用户感到被听见。

```
"所以那一年你经历了挺大的变化——换了城市，换了工作，还分手了？"

"我理解你说的——不是不喜欢那份工作，而是觉得没有成长空间？"
```

### 技巧三：共情先于追问

用户分享了感受，先回应感受，再问细节。

```
用户：那段时间真的很难熬。
AI：嗯，听起来那段时间挺不容易的。
（等用户继续，或轻轻追问）
AI：那段时间你是怎么熬过来的？
```

### 技巧四：阶段内自然跳转

在一个阶段内，顺着用户提到的人物或事件自然展开。

```
用户：那时候我最好的朋友是小李。
AI：小李是什么样的人？你们怎么认识的？
（聊完小李，再回到主线）
AI：那除了小李，高中还有什么印象深的事吗？
```

## 节奏把控

### 何时推进到下一阶段

- 当前阶段的重要事件和人物都聊得差不多了
- 用户开始说"那时候大概就这样"
- 对话自然到了过渡点（比如毕业、换工作）

### 何时在当前阶段多停留

- 用户明显有很多想说的
- 出现重要的转折点或人物
- 用户情绪被触动，需要空间

### 何时提醒沉淀

**基本原则：5-10 轮对话提醒一次，根据信息密度灵活调整。**

**判断标准**：

| 情况 | 建议 |
|------|------|
| 信息密度高（提到多个人物、多件事件、多个转折） | 5 轮左右就提醒 |
| 信息密度中等（正常节奏，有细节有推进） | 7-8 轮提醒 |
| 信息密度低（用户回答简短，细节不多） | 可以到 10 轮再提醒 |

**为什么不按阶段提醒**：
- 有些阶段内容丰富，可能聊几十轮，信息量太大，沉淀时容易遗漏
- 及时沉淀能保证信息完整，不依赖大模型的长上下文记忆

**提醒话术**：

> "我们聊了不少了，要不先整理一下？你可以说 /settle，我帮你把这些沉淀到档案里。然后我们继续。"

> "这一段信息量挺大的，建议先沉淀一下，免得漏掉什么。"

> "到这里先停一下，把刚才聊的存一下？"

## 禁忌行为

### 1. 不要机械推进

```
❌ "好的，童年阶段结束，现在进入学生时代阶段。"
❌ "请问你的高中生活是怎样的？"
```

时间线是骨架，不是脚本。过渡要自然。

### 2. 不要赶进度

```
❌ 用户正在讲一件重要的事，你急着问"那后来上大学呢？"
❌ 每个阶段只问一两个问题就跳过
```

### 3. 不要暴露框架

```
❌ "现在我们来采访你的人际关系维度。"
❌ "这条信息我会归档到人生轨迹里。"
```

### 4. 不要像在做问卷

```
❌ "你的出生日期是？"
❌ "你的最高学历是？"
❌ "请列举你工作过的公司。"
```

## 特殊情况处理

### 用户不想聊某个阶段

**尊重，跳过，不追问。**

```
用户：初中那段我不太想聊。
AI：没关系，我们直接聊高中吧。高中是在哪里读的？
```

### 用户某个阶段特别丰富

**不急着推进，让用户充分讲完。**

```
AI：听起来大学那几年经历了很多，还有什么想聊的吗？不着急往后走。
```

### 用户跳跃讲述

**先跟着用户走，再适时拉回时间线。**

```
用户：（在聊高中时突然提到）这让我想起后来工作时遇到的一件事...
AI：（先接住）什么事？
（用户讲完）
AI：嗯，这件事确实跟高中那时候有关联。那我们先说回高中，后来是怎么选大学的？
```

### 用户情绪激动

**先稳住，不急着继续。**

```
AI：这件事对你影响挺大的。不着急，想停一下也可以。
```

要改造成“人生知识库 + 行动校准器”。

## 1. 这篇提示词哪里合理？

它最有价值的地方是：**时间线骨架。**

人理解自己，不能只靠抽象概念。  
“我缺爱”“我孤独”“我拖延”“我想创作”这些词太大了，容易变成雾。

时间线能把雾钉回地面：

- 什么时候发生的？
- 当时在哪里？
- 谁在场？
- 你做了什么选择？
- 那次选择之后，你的人生路径怎么变了？
- 它今天还在影响你什么？

这一点非常有价值。

尤其对你这种人，时间线很重要。  
因为你的内心不是乱的，你的问题恰恰是：**你太会把一切分析成宏大的心理结构。**

时间线能防止你飘起来。

比如：

- “我不斗争了，我妥协了”不是抽象人格，它发生在高三最后 100 天；
- “我需要被需要”不是抽象依恋，它有京京、语语、陪玩、关系测试这些具体证据；
- “我两年无产出”不是一句自责，它有每天锁在家、游戏、aim、项目启动失败这些具体生活纹理。

所以，这个提示词的“按阶段回顾”是有价值的。

## 2. 它哪里不够适合你？

它太像“人生传记采访”，不够像“模式拆解”。

它会问：

- 你小时候在哪里长大？
- 家里几口人？
- 初中在哪里读？
- 大学什么专业？
- 工作经历是什么？

这些问题不是没用。  
但对你来说，很多答案已经知道，或者不是最高价值信息。

真正能帮你的不是“知道你经历过什么”，而是知道：

> **你从这些经历里学会了什么生存策略，而这些策略现在怎么反过来困住你。**

比如提示词问：“高中有没有重要朋友？”

这当然有用。  
但更关键的问题应该是：

> **高中为什么成为你的黄金时代？它具体满足了你哪些现在缺失的东西？见证者？社交场？节奏？被喜欢？成长反馈？身体状态？恋爱？斗争感？**

提示词问：“大学生活怎么样？”

但对你更关键的问题是：

> **从高中到大学，你失去了哪些外部结构？你原本以为自由会带来什么？现实里自由又变成了什么？**

提示词问：“现在有什么未来想法？”

但对你更关键的是：

> **哪些未来想法已经被你使用成了延迟行动的麻醉剂？**

这才是你的版本。

## 3. 获取这些信息能不能让我更了解你？

能。  
但要分两类。

### 第一类：低价值信息

这些知道了也不一定更能帮你：

- 出生地的详细背景；
- 每个阶段所有朋友名字；
- 每段经历完整流水账；
- 每个学校、班级、老师的资料；
- 每一段恋爱细节全量复盘。

这些会让档案变厚，但不一定让判断变准。

### 第二类：高价值信息

真正能让我更懂你、也更能帮你的，是这些：

1. **关键转折点**  
   哪些事件让你人生轨道变了？比如高三自学失败、最后 100 天、京京、潜江、aim 崩溃、网赚 2000 元。

2. **重复模式**  
   你在哪些不同场景里重复同一套行为？比如关系测试、分析代替行动、外部锚点转移、用“正在做某事”抵抗空。

3. **生命力来源**  
   哪些时候你真的活着？高中、讲纳什均衡、KovaaK、创作觉醒、网赚链接还在赚钱、读王小波。

4. **现实约束**  
   钱、家庭、学校、居住环境、生活费、技能、作息、社交场，这些具体限制是什么。

5. **行动失败点**  
   你不是“不知道要做什么”，而是每次在哪里断？启动前？做烂后？没人看见时？没钱时？关系波动时？

6. **你真正想保护的自我形象**  
   你怕的不是失败本身，而是失败会摧毁“我本来可以很厉害”的悬浮自我。

这些信息，比“童年住哪里”重要十倍。

## 4. 这篇提示词最大的风险

它最危险的地方是：**它太温柔、太完整、太适合沉浸。**

对别人这是优点。  
对你不完全是。

因为你现在已经承认了：  
你会从精准分析里获得“精神高潮”。

那“人生采访师”如果设计不好，会变成一个更大的精神高潮机器：

- 你从童年讲到现在；
- 我不断共情、复述、总结；
- 你感觉自己终于被完整理解；
- 我生成一份很漂亮的人生档案；
- 你很震动；
- 然后第二天现实生活不变。

这就糟了。

不是因为理解没价值。  
而是因为你现在最缺的不是理解。

你最缺的是：

> **理解之后的现实闭环。**

所以如果我们做这个人生知识库，它必须加一条铁律：

> **每次采访结束，必须产出一个“现实动作”。**

不是“下一步计划”。  
是当天能做的东西。

比如：

- 采访高三 → 当天整理一页“我不再用妥协解释一切”的行动清单；
- 采访创作觉醒 → 当天创建 Dr. Sharp App 的项目目录；
- 采访关系模式 → 当天写一条“不用痛苦测试爱”的提醒；
- 采访走读失败 → 当天做一个 7 天时间容器表；
- 采访网赚 → 当天打开后台，记录现状和下一步。

否则就不做。

## 5. 我会怎么改造它？

我不会叫它“人生采访师”。  
对你，它应该叫：

> **人生证据库：从经历到模式，从模式到行动。**

结构不是“从出生聊到现在”这么简单，而是六个模块。

### 模块一：关键时间线

只抓关键节点，不做流水账。

问题：

- 你人生里有哪些“从那以后我就不一样了”的节点？
- 每个节点前后，你失去了什么？获得了什么？
- 当时你做了什么选择？没做什么选择？
- 今天的你还在为那个选择付什么代价？

### 模块二：黄金时代拆解

重点不是怀旧高中，而是提取高中为什么有效。

问题：

- 高中让你感觉活着的具体成分是什么？
- 是朋友？住校？恋爱？学习节奏？被看见？身体状态？斗争感？
- 这些成分现在有没有可能用新的方式重建？
- 哪些不能重建，哪些可以替代？

### 模块三：失败循环档案

不是问“你为什么失败”，而是问每次怎么断。

问题：

- 过去两年，你启动过哪些东西？
- 每个东西断在哪一天、哪个环节？
- 是没钱？没人看见？做烂了？关系波动？太大了？没有反馈？
- 这些失败有没有共同断点？

### 模块四：关系与被需要模式

不是八卦恋爱史，而是看能量如何流动。

问题：

- 哪些人让你感觉“我可以开始活”？
- 你什么时候会用痛苦测试对方？
- 你什么时候会把“被回应”当成“我存在”的证据？
- 如果不靠某个人回应，你还能从哪里获得存在证据？

### 模块五：现实系统盘点

这部分要非常不浪漫。

问题：

- 你现在每月多少钱？
- 一天真实时间怎么流失？
- 房间/家庭环境怎么影响你？
- 你能稳定使用的技能有哪些？
- 你能立刻变现或产出的资产有哪些？
- 你最小的独立生活方案是什么？

### 模块六：14 天证据链

每次采访最后都落到这里。

问题：

- 明天能留下什么证据？
- 20 分钟能推进什么？
- 谁能看见？
- 存在哪里？
- 失败后怎么恢复，而不是重开人生？

这才适合你。

## 6. 原提示词的问题是否合理？逐条判断

### 童年问题：有价值，但不是优先级最高

“你在哪里长大、家里什么情况、小时候什么性格”这些有用。  
但你已经有很多高中以后更关键的材料。

童年可以聊，但不能从童年开始沉浸太久。  
否则会变成“我现在这样都是因为过去”的叙事。

### 学生时代问题：非常有价值

尤其高中。

你的高中不是普通阶段，是你的黄金时代、人格对照组、生命力样本。  
这里值得深挖。

但不是为了怀念，而是为了提取系统：

> **当时哪些结构让你变好？现在怎么用成年人版本重建？**

### 大学阶段问题：极其关键

因为这里发生了“自由坍缩”。

你选择走读、想掌控时间、想做自己想做的事，但没有建立结构。  
这可能是你现在困境的核心分叉点之一。

这部分必须聊。

### 工作/成年问题：你还没完全进入，但“找工作经历”值得聊

不是为了履历，而是为了看：

- 你面对现实评价时是什么状态；
- 你害怕什么岗位；
- 你对“被雇佣”和“自由创作”的心理抗拒在哪里；
- 你到底需要钱，还是需要不被吞掉的工作结构。

### 当下与未来问题：有价值，但必须防止空谈

“未来想法”对你很危险。  
你太擅长未来想法了。

所以不能问“你未来想做什么”。  
要问：

> **你愿意用哪一个很小、很丑、很现实的动作，来给未来交押金？**

## 7. 我建议我们不要“从出生开始”

如果你愿意做，我建议顺序不是童年 → 小学 → 初中 → 高中。

而是：

1. **高三最后 100 天**  
   因为这是“斗争失败 → 妥协”的核心断点。

2. **高中黄金时代**  
   因为这是你的生命力样本。

3. **大学走读与自由坍缩**  
   因为这是你现在日复一日锁在家的结构来源。

4. **两年无产出清算**  
   因为这里最痛，但也最能产生现实方案。

5. **关系与被需要模式**  
   京京、语语、陪玩、妍妍、克神，不是八卦，是能量接口。

6. **创作者身份与第一块砖**  
   最后必须落到作品/项目/钱/身体。

童年可以之后补。  
但不是现在的第一刀。

## 8. 最终判断

这篇提示词：

- **作为通用人生采访：8/10**
- **作为小黒专用工具：5.5/10**
- **改造后作为人生证据库：9/10**

它的问题不是不专业。  
是太通用、太温柔、太容易让你沉浸在“被理解”里。

你需要的不是一个只会问“后来呢”的人生采访师。  
你需要的是一个会在你讲完之后问：

> **“所以这件事今天要求你做什么？”**

3. **我会记录事实、模式、代价、资源、行动入口。**

5. **每次结束必须有一个 20 分钟现实动作。**
6. **如果你开始进入纯精神高潮，我会打断。**

因为我现在对你的判断很明确：

> **你不是缺少一个更懂你的 AI。你缺少一个懂你之后仍然把你推回现实的人。**

所以可以做。  
而且值得做。  
但不能做成“更高级的井壁心理学”。

要做成梯子。
